## How to add new cats


1. Get your favorite Cat photo (Please only use images where you have the rights to the images)
No AI images allowed in this project. Real :cat: only

2. Add LGTM to the image. Feel free to use this [Canva](todo) template. Ensure good color contrast for accessibility. Try to keep the image size under 1MB. You can use Canva or imagemagik to resize.

3. Clone this repo

4. Add your image to the `/images` folder

5. Copy the following markdown and add to this README

```
![LGTM Cat Preview](https://github.com/ann-kilzer/LGTM-cat/images/TODO)

\```
[![LGTM Cat](https://github.com/ann-kilzer/LGTM-cat/images/TODO)](https://github.com/ann-kilzer/LGTM-cat)
\```
```


6. Open a PR
